Stack structure implementation in Python
Stack is one of the data structure that can implement by array or linked list.
So, two different class is provided that create stack object with its properties and methods.
There are three .py files:

--stack_array.py :   Python program to demonstrate stack implementation using a array.
--stack_linked_list: Python program to demonstrate stack implementation using a linked list 
		     (Refference: https://www.geeksforgeeks.org/stack-in-python/).
--test.py:           Test file for objects implementation.
